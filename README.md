# zero_waste_app

Aplicación de flutter desarrollada para reto de FractalUp 
Elaborado por: Marcelo Alessandro Varela Bustinza - VarBus

Descripción: 
ZeroWaste es una aplicación desarrollada en Flutter enfocada en la gestión inteligente de alimentos, cuyo objetivo principal es reducir el desperdicio y mejorar los hábitos de consumo mediante el uso de tecnología.
